# Rekgizi Frontend

Frontend Rekgizi menggunakan Vue 3, TypeScript, Vite, dan Tailwind CSS.

```powershell
npm ci
Copy-Item .env.example .env
npm run dev
```

Isi `VITE_API_URL=http://127.0.0.1:8000` di `.env`. Konfigurasi Firebase bersifat opsional jika push notification tidak digunakan.

Konfigurasi environment dan daftar library lengkap tersedia di [dokumentasi setup](../README.md).
